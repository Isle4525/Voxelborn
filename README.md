## 🎮 Геймплей

Вы — отважный искатель приключений, спускающийся в глубины постоянно меняющегося подземелья. Каждый уровень уникален и таит в себе новые опасности.

### Классы героев
Выберите свой стиль игры:
*   🗡️ **Воин** — Высокая защита и мощные удары ближнего боя. Навыки: *Удар щитом*, *Рывок*.
*   🔮 **Маг** — Мастер заклинаний с огромным уроном, но малым запасом здоровья. Навыки: *Огненный шар*, *Ледяная стрела*.
*   🏹 **Разбойник** — Мастер критических ударов и скорости. Навыки: *Смертельный удар*, *Двойной удар*.

### Мир игры
Исследуйте разнообразные биомы, каждый со своим визуальным стилем:
*   🌑 **Пещеры** (#) — классические каменные залы.
*   🌲 **Леса** (^) — заросшие густой растительностью тропы.
*   🔥 **Лавовые озера** (%) — жаркие и опасные глубины.
*   🏜️ **Пустыни** (+) — выжженные пески.
*   ❄️ **Ледяные пустоши** (*) — замерзшие коридоры.

## 🛠️ Управление

Игра оптимизирована для игры на клавиатуре:
*   `W`, `A`, `S`, `D` — Передвижение по карте и атака врагов.
*   `1`, `2` — Использование уникальных навыков класса.
*   `U` — Выпить зелье здоровья (восстанавливает 10 HP).
*   `M` — Выпить зелье маны (восстанавливает 10 Mana).
*   `Q` — Выход из игры.

## 📸 Интерфейс (Пример)

```text
#################################  =============================
#.......G.......................#  Уровень: 1
#...............................#  Биом: Пещера
#......@..........!.............#  
#...............................#  HP: 30/30
#...............................#  Мана: 10/10
#...........>...................#  Уровень: 1
#...............................#  Опыт: 0/100
#################################  =============================
```
*`@` - Вы, `G` - Гоблин, `!` - Зелье, `>` - Спуск на следующий уровень.*

## 🚀 Запуск

1. Убедитесь, что у вас установлен **Python 3.x**.
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/Voxelborn.git
   ```
3. Запустите игру:
   ```bash
   python main.py
   ```
